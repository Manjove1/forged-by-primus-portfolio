# Combiners

This folder contains premade combiner teams and their component characters.

**Design Rule:**  
Combiner teams are treated as a unit by default. When a combiner is relevant, prefer loading the dedicated combiner file. Individual components may appear alone only in specific circumstances (separation, capture, solo missions, etc.).

Universal rules live in `core-systems/COMBINER_MECHANICS.md`.

## File Contents

Each combiner file contains:
- The combined form entry
- All component characters
- Combination and separation guidance
- Team-specific tension notes

Components that have strong independent presence may also have light cross-references in their normal faction category files.
