# SillyTavern Import

## Lorebook package

File: `forged-by-primus-core-systems.json`

### Import steps

1. Open SillyTavern → **World Info** (Lorebook)
2. Click **Import**
3. Select `forged-by-primus-core-systems.json`
4. Attach the lorebook to your chat or character
5. Start a new chat
